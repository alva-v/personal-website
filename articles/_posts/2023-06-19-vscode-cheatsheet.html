Some shortcuts and config options I too often forget.
I use all of these on VSCodium Linux.

Shortcuts

line wrap
Alt+Z
format
Ctrl+Shift+I

Config options

editor.rulers

Displays one or more ruler at x characters from the line start. {% highlight json %} "editor.rulers": [ 80, 120 ]{% endhighlight %}

files.associations

Associate file extensions with language modes by language identifiers. {% highlight json %} "files.associations": { "*.html": "liquid", "*.md": "liquid", "**/assets/css/*.scss": "liquid-scss", "**/_sass/*.scss": "scss" }{% endhighlight %}

Shortcuts

line wrap

format

Config options

editor.rulers
Displays one or more ruler/vertical line at x characters from the left. Value is an int array of the number of character from which to display the ruler. {% highlight json %} {% raw %} "editor.rulers": [ 80, 120 ] {% endraw %} {% endhighlight %}
files.associations
Associate file extensions with language modes by language identifiers. {% highlight json %} {% raw %} "files.associations": { "*.html": "liquid", "*.md": "liquid", } {% endraw %} {% endhighlight %}
